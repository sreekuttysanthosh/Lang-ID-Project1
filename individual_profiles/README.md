# Individual Contribution Profiles

Each team member must add their GitHub activity screenshot here.

## Instructions
1. Go to your GitHub profile -> repository -> Insights -> Contributors
2. Screenshot your commits, additions, deletions
3. Save as [your_name]_github_activity.png in this folder
